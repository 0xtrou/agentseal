# Configuration

## Environment Variables

| Variable | Description |
|----------|-------------|
| `SNAPFZZ_SEAL_MASTER_SECRET_HEX` | 64-hex master secret |
| `SNAPFZZ_SEAL_LAUNCHER_PATH` | Launcher binary path |
| `SNAPFZZ_SEAL_LAUNCHER_SIZE` | Launcher size for self-extraction |
| `RUST_LOG` | Log level (debug, info, trace) |
